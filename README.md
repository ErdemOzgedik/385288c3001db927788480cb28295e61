# Shopify App

> https://shopifyyyy.herokuapp.com

**To Run On Local**

## `npm install`

Installs required packages.

> Way 1

# `npm run dev`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

> Way 2

# `npm run build`

Creates a build folder.

## `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
